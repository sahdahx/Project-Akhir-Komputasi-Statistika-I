## Statistical Inference Dashboard Using R Shiny

This project aims to create an **interactive dashboard for statistical inference analysis** using **R Shiny**.  
The dashboard allows users to perform hypothesis testing, descriptive analysis, and data visualization interactively.

Public Dashboard Link: (sahdahuwaidah.shinyapps.io/Inferensii)[sahdahuwaidah.shinyapps.io/Inferensii]

---

## ðŸ“Œ Main Features

1. **One-Sample Test**
   - Upload your dataset
   - Define hypotheses and significance level
   - Get test statistics, critical region, conclusion, and automatic interpretation

2. **Two Independent Samples Test**
   - Select columns for each population
   - Check assumptions: normality and equal variance
   - Perform t-test for independent samples
   - Results include tables, histograms, test statistics, and automatic interpretation

3. **Two Dependent Samples Test**
   - Test normality of differences and paired t-test
   - Histogram of differences
   - Automatic interpretation based on test results

4. **One-Way ANOVA**
   - Upload response and factor variables
   - ANOVA results: F-test table, boxplot, residual plots, and interpretation
   - Assumption tests: Shapiro-Wilk (normality of residuals) and Bartlett (homogeneity of variance)
   - Full interpretation based on p-values
